# Portfolio App
## Summary
Simple app to aggregate stock positions

## Stack
### Django
- Django Rest Framework - server
- Django Rest Knox - authentication

### React
- React Router
- React Redux
- Material UI
- axios

### Dev
 - Webpack 
 - Babel

### Third Party API
- IEX 

## QuickStart

`cd Portfolio_App`

`python App_Manager/manage.py runserver`

`cd frontend`

`npm install`

`npm run dev`
